这是一个由个人制作的黄金价格分析工具，主要是体现黄金价格变化趋势，展示各类价格参数，便于分析黄金价格变化，个人买卖黄金时可用于参考。

主要功能：  
 1.黄金历史价格查询和价格极值分析  
 2.黄金买卖建议  
 3.黄金交易收益试算
